# Scyrox V6 Battery Monitor

A Python-based system tray utility to monitor the battery level of your Scyrox V6 mouse. 

## Features
- Real-time battery monitoring directly from your system tray.

## Requirements
To install the required dependencies, run:

```bash
pip install -r requirements.txt
```

## Usage
Simply run the python script:

```bash
python scyroxv6batteryontray.pyw
```
Or double-click the `scyroxv6batteryontray.pyw` file to start it in the background.
